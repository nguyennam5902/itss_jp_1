# itss_jp_1
ITSS Japanese 1
### Installation
Step 1: Clone repository <br/>
```shell
git clone https://github.com/nguyennam5902/itss_jp_1.git
```
Step 2: Go to backend folder<br/>
```shell
cd .\itss_jp_1\backend
```
Step 3: Install dependency<br/>
```shell
npm ci
```
Step 4: Run<br/>
```shell
node .\index.js
```
